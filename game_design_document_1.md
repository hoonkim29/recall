## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **게임 이름** | Recall |
| **장르** | 로그라이크 덱빌딩 + 방탈출 + 메타 호러 |
| **플랫폼** | 모바일 (iOS / Android) |
| **엔진** | Godot 4 (GDScript) |
| **해상도** | 1080 × 1920 (세로형 모바일) |
| **수익 모델** | 챕터 언락 (Freemium) + 코스메틱 |
| **타겟** | 전 세계 |

### 이름의 의미
| 레이어 | 의미 |
|--------|------|
| 표면 | "기억을 떠올리다" |
| 중간 | "Re-call / 반복 소환" — 서고가 플레이어를 계속 불러들임 |
| 진실 | "제품 리콜" — 결함 있는 기억 수집 도구를 회수 |

## 3. 타이틀 화면

**배경**: 어두운 기숙사 방 (정적 이미지 or ColorRect 플레이스홀더)
**카드 덱**: 책상 위 중앙. 탭하면 게임 시작.
**UI 요소**:
- 게임 이름 "RECALL" (상단 중앙)
- 카드 덱 오브젝트 (탭 가능)
- 🔖 컬렉션 버튼 (하단 좌측) — 데모에선 비활성
- 🛍 상점 버튼 (하단 우측) — 데모에선 비활성

**회귀 후 타이틀**: 죽으면 "눈을 뜸 → 핸드폰 집어듦 → 앱 켬" 짧은 페이드 후 타이틀 복귀.

## 4. Room 1 Path 구조 (상세)

```
[시작]
    │
    ├── A노드: ⚔ 일반 배틀
    └── B노드: 📦 기억 상자 #1
              │
    ┌─────────┴──────────┐
    │                    │
C노드: 🔥 캠프파이어    D노드: ⚔ 일반 배틀
    │                    │
    └─────────┬──────────┘
              │
         E노드: 📦 기억 상자 #2
              │
         F노드: ⚔ 일반 배틀
              │
         [자물쇠 화면] ── 힌트 3개로 자물쇠 3개 해제
              │
         G노드: 👁 ELISE (보스)
              │
         [LOOP ENDING or 방 2 진입]
```

**노드 상세**:
- 일반 배틀: Elise 덱의 약한 버전과 싸움. 이기면 카드 1장 선택 획득 또는 힌트 조각.
- 기억 상자: 배틀 없이 바로 열림. 힌트 파편 1개 획득.
- 캠프파이어: 카드 1장 공격력 +1 강화 OR 조각 1개 회복 중 선택.
- 자물쇠 화면: 3개 자물쇠. 힌트로 해제. 전부 해제해야 보스 진입 가능.
- Elise 보스: 풀 덱으로 배틀. 이기면 방 클리어.

## 5. 자물쇠 퍼즐 상세 (Room 1)

### 자물쇠 1 — 날짜 코드 (4자리 숫자)
**정답**: 1103
**힌트 위치**: 기억 상자 #1
**힌트 내용**:
> *찢긴 입고 기록부 한 페이지. "입고일: 11월 03일" 이라고 적혀있다. 이름란은 찢겨 있다.*

### 자물쇠 2 — 이름 코드 (5글자)
**정답**: ELISE
**힌트 위치**: 일반 배틀 승리 보상 (E노드 이전 배틀)
**힌트 내용**:
> *오래된 명찰 조각. "E _ I _ E" 라고 적혀있다. 가운데 글자들이 지워져 있다.*

**교차 힌트**: 캠프파이어에서 추가 힌트 —
> *수첩 메모: "접수실 사서. 안경. 머리카락이 짧은 여자. 이름이 E로 시작함."*

### 자물쇠 3 — 기호 코드 (기호 2개 순서)
**정답**: ◆ → ▲ (순서 맞춰야 함)
**힌트 위치**: 기억 상자 #2
**힌트 내용**:
> *그림이 그려진 메모지. 다이아몬드 다음에 삼각형이 그려져 있다. 아래에 흐릿하게 "순서대로" 라고 적혀있다.*

**교차 힌트**: 카드 "길 잃은 기억"의 플레이버 텍스트에 ◆ 기호가 있음 — 눈썰미 있는 플레이어만 발견.

## 6. 카드 시스템 상세

### 자원 시스템
| 자원 | 획득 | 소비 |
|------|------|------|
| **잉크 (Ink)** | 매 턴 시작 시 3개 자동 지급. 턴 종료 시 잔량 소멸 | 카드 내는 데 사용 |
| **망각 (Oblivion)** | 플레이어가 덱에서 카드 1장을 영구 제거할 때 발동 | 잉크 +2 획득 (이번 턴만) |
| **기억 조각 (Fragment)** | 시작 시 양쪽 모두 5개 | 카드 공격받으면 수호자 조각 빼앗김. 내 조각 0 → 패배 |

### 턴 구조 (상세)
```
[플레이어 턴]
1. 턴 시작: 잉크 3개 지급, 덱에서 카드 1장 드로우
2. 행동 페이즈:
   a. 손패에서 카드 선택 → 잉크 소비 → 슬롯(1~4)에 배치
   b. 원하면 망각 선택: 덱에서 카드 1장 제거 → 잉크 +2 즉시 획득
   c. 원하는 만큼 카드 배치 (잉크 허용 범위 내)
3. 전투 페이즈: 내 슬롯 카드들이 맞은편 슬롯 수호자 카드를 공격
   - 공격력 > 방어력: 차이만큼 수호자 조각 빼앗음
   - 공격력 ≤ 방어력: 수호자 조각 변화 없음
   - 맞은편 슬롯이 비어있으면: 공격력만큼 수호자 조각 직접 빼앗음
4. 수호자 턴: 수호자가 카드를 내고 공격
   - 수호자 카드가 내 슬롯 공격
   - 내 슬롯이 비어있으면: 공격력만큼 내 조각 빼앗음
5. 양쪽 조각 확인:
   - 수호자 조각 0 → 플레이어 승리
   - 플레이어 조각 0 → 패배 → 회귀

[슬롯 규칙]
- 슬롯 4개 (왼쪽부터 1,2,3,4)
- 슬롯 1 ↔ 수호자 슬롯 1, 슬롯 2 ↔ 수호자 슬롯 2 ... 정면 대결
- 한 슬롯에 카드 1장만 배치 가능
- 턴이 끝나도 카드는 슬롯에 남아있음 (파괴되거나 능력으로 제거되기 전까지)

[망각 선택 UI]
- "망각" 버튼 탭 → 덱 카드 목록 팝업
- 플레이어가 제거할 카드 선택
- 확인 → 카드 영구 제거, 잉크 +2
- 회귀 시 망각된 카드 전부 복구 (공격력 +1 "기억된" 버전으로)
```

### 카드 전체 수치 확정

**시작 덱 (10장)**
| 카드명 | 잉크 | 공격 | 방어 | 인장 | 비고 |
|--------|------|------|------|------|------|
| 흐릿한 기억 | 0 | 0 | 0 | 망각 재료 | ×4. 망각 시 잉크 +2 |
| 잔상 | 1 | 1 | 0 | — | ×2 |
| 기억의 파편 | 1 | 0 | 2 | 파쇄 | ×2. 파괴 시 조각 1개 회수 |
| 메아리 | 2 | 2 | 1 | — | ×1 |
| 길 잃은 기억 | 1 | 0 | 1 | 대화 | ×1. 말하는 카드 |

**획득 가능 카드 (Room 1 배틀 보상)**
| 카드명 | 잉크 | 공격 | 방어 | 인장 |
|--------|------|------|------|------|
| 야경꾼 | 1 | 1 | 1 | 감시: 공격받으면 공격자 -1 공격력 |
| 서고 쥐 | 0 | 1 | 0 | 덧없음: 공격 후 자동 파괴 |
| 낡은 책 | 1 | 0 | 3 | 기록: 3턴 생존 시 조각 1개 획득 |
| 잊혀진 개 | 2 | 2 | 2 | — |
| 반향 | 1 | 0 | 1 | 반향: 공격받으면 조각 1개 반환 |

### Elise 덱 구성 (15장)

**일반 배틀 (약한 버전 - 10장)**
| 카드명 | 잉크 | 공격 | 방어 | 수량 |
|--------|------|------|------|------|
| 흐릿한 기억 | 0 | 0 | 0 | 3 |
| 잔상 | 1 | 1 | 0 | 3 |
| 야경꾼 | 1 | 1 | 1 | 2 |
| 기억의 파편 | 1 | 0 | 2 | 2 |

**보스 배틀 (풀 덱 - 15장)**
| 카드명 | 잉크 | 공격 | 방어 | 수량 |
|--------|------|------|------|------|
| 흐릿한 기억 | 0 | 0 | 0 | 3 |
| 잔상 | 1 | 1 | 0 | 3 |
| 야경꾼 | 1 | 1 | 1 | 3 |
| 기억의 파편 | 1 | 0 | 2 | 2 |
| 잊혀진 개 | 2 | 2 | 2 | 2 |
| 메아리 | 2 | 2 | 1 | 2 |

**Elise AI 행동 패턴**
1. 잉크가 허용되면 공격력 높은 카드 우선 배치
2. 플레이어 조각이 2개 이하면 공격형 집중
3. 3턴마다 반드시 방어 카드 1장 배치
4. 빈 슬롯에 우선 배치

## 7. 회귀 시스템 (상세)

**죽을 때 (조각 0)**:
1. 화면 페이드 아웃
2. 짧은 컷씬: 기숙사 방, 눈 뜸, 핸드폰 켬
3. 타이틀 화면으로 복귀

**유지되는 데이터 (세이브)**:
- 회귀 횟수 (run_count)
- 이전 런에서 망각된 카드 목록 → 다음 런에서 "기억된" 버전으로 복구
- 해제된 자물쇠 (solved_locks 배열) — 이미 푼 자물쇠는 다음 런에서 열린 상태

**리셋되는 데이터**:
- 현재 덱 (시작 덱으로 초기화, 단 기억된 카드 포함)
- 현재 조각 수
- Path 진행 상태
- 힌트 수집 상태

**기억된 카드 (Remembered Card)**:
- 이전 런에서 망각한 카드가 다음 런 시작 덱에 포함됨
- 공격력 +1 버전으로 복구
- 카드 이름 앞에 "[기억된]" 표시
- 최대 3장까지 누적

## 10. LOOP ENDING

Elise를 이긴 후:
1. 화면 페이드
2. 텍스트:
   > *"탈출했다. 기숙사 방. 핸드폰이 울린다.*
   > *J한테서 온 메시지: "야, 나 어제 왜 연락 안 받았어?"*
   > *평범한 아침. 모든 게 끝난 것 같다.*
   > *...*
   > *그런데 핸드폰 화면에 Recall 알림이 떠있다.*
   > *"새로운 기억이 기록되었습니다."*
3. 타이틀 화면으로 복귀 (run_count +1)
