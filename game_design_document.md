## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **게임 이름** | Recall |
| **장르** | 로그라이크 덱빌딩 + 방탈출 + 메타 호러 |

### 이름의 의미

| 레이어 | 의미 |
|--------|------|
| 표면 | "기억을 떠올리다" — 따뜻하고 노스탤직한 퍼즐 게임처럼 들림 |
| 중간 | "Re-call / 반복 소환" — 서고가 플레이어를 계속 불러들이는 구조 |
| 진실 | "제품 리콜" — 결함 있는 기억 수집 도구를 회수하는 것 |

처음엔 "기억 게임인가보다" → 게임을 다 하고 제목 화면을 다시 보는 순간 → **"아, 내가 소환당하고 있었구나"**

## 2. 핵심 컨셉

카드 배틀로 얻은 힌트로 방탈출 자물쇠를 열고, 죽으면 기억을 갖고 회귀하는 주인공.

### 3중 스토리 레이어

#### 레이어 1 — 표면 (처음부터 보임)
앱을 실행하니 낯선 서고에 갇혀있다. 카드 게임으로 수호자들을 이기고 방을 탈출하자.

#### 레이어 2 — 중간 진실 (방 2~3에서 드러남)
이 서고는 사라진 사람들의 기억을 보관하는 곳. 수호자들은 기억을 빼앗긴 채 강제로 묶여있는 사람들이다. 내가 모으는 문서 조각들은 그들의 기억이다.

> "나는 그들을 해방시키고 있는 건가, 아니면 기억을 훔치고 있는 건가?"

#### 레이어 3 — 진짜 반전 (마지막 방 + 숨겨진 엔딩)
플레이어 자신도 기억이 일부 봉인된 상태였다. 이 앱을 만든 창조자도 서고에 갇혀 있으며, 앱 자체가 탈출 신호다. 진짜 엔딩은 타이틀 화면의 비밀 상자를 열어야 한다.

### 핵심 반전 — 회귀 메커니즘의 정체

- **플레이어 입장**: "내 의지로 기억을 갖고 회귀한다" → 점점 강해진다
- **서고 입장**: 더 많은 기억을 수집하기 위해 플레이어를 반복 소환한다

> 회귀할수록 강해지는 주인공 = 서고의 데이터가 쌓일수록 강해지는 적

## 3. 오프닝 + 타이틀 구조

### 최초 실행 영상 (첫 실행 시 한 번만)

**장면 1 — 낮. 캠퍼스.**
J와 걷고 있다. J가 핸드폰을 꺼내 보여준다.
> *"나 요즘 이거 하는데 진짜 신기해. 너도 해봐."*

Recall 앱. 주인공이 흘깃 본다. 별 관심 없는 표정.

**장면 2 — 그날 밤. 기숙사 방.**
핸드폰으로 앱스토어에서 Recall 검색. 다운로드. 켜지 않음. 잠.

**장면 3 — 며칠 후. 밤.**
J한테 연락이 안 된다. 문자, 전화. 읽씹. 부재중.
멍하니 핸드폰을 보다 홈화면의 Recall 아이콘. 누르는 클로즈업.

**→ 타이틀 화면으로 전환.**

---

### 타이틀 화면

```
[핸드폰 화면 속 기숙사 방 — 밤]
현실의 기숙사 방과 동일한 공간.
단 하나 다른 것 — 책상 위 카드 한 벌.

              RECALL

        🔖            🛍
      컬렉션          상점
```

- **게임 시작 버튼 없음** — 카드 덱을 탭하면 게임 시작
- **이어하기 없음** — 죽으면 항상 타이틀로 돌아옴 (회귀 구조)
- **컬렉션**: 코스메틱 관리 및 장착, 오버레이 조합
- **상점**: 챕터 패스, 코스메틱 팩 구매

### 타이틀 배경의 변화 (회귀 횟수에 따라)

현실의 방은 변하지 않음. 핸드폰 속 방(타이틀 배경)만 조금씩 달라짐.

| 회차 | 핸드폰 속 방 변화 |
|------|----------------|
| 1~2회차 | 깔끔함. 카드 덱만 있음 |
| 3회차 | 책상 모서리에 살짝 보이는 무언가. 탭 영역 없음 |
| 5회차 | 작은 나무 상자 윤곽이 흐릿하게 보임. 탭해도 반응 없음 |
| 7회차 | 상자가 완전히 나타남. 마치 원래 있었던 것처럼 |
| 10회차+ | 상자 다이얼이 미세하게 빛남 |
| TRUE ENDING 직전 | 현실의 방에도 상자가 생겨있음 (역전) |

### 죽은 후 타이틀 복귀

게임 안에서 죽으면 — 짧은 컷씬.
기숙사 방. 눈을 뜸. 핸드폰을 집어듦. Recall을 켬.
핸드폰 화면으로 줌인 → 타이틀 화면.

## 4. 비밀 장치 — 다이얼 상자

### 개요

타이틀 화면 기숙사 방 책상 위에 회차가 쌓이면서 서서히 나타나는 작은 나무 상자. 별도 자물쇠 없이 **상자 뚜껑 표면에 다이얼 4개가 직접 새겨져** 있음.

처음엔 장식 문양처럼 보임. 가까이 탭해서 클로즈업됐을 때 돌아가는 다이얼이라는 것을 알게 됨.

### 다이얼 배치

뚜껑 표면에 불규칙하게 흩어진 패턴으로 배치 — 일렬 배치보다 장식처럼 보이게.

```
  ◇     ◇
     ◇
  ◇
```

각 다이얼은 숫자나 알파벳이 아닌 **기호**가 새겨져 있음. 코스메틱 오버레이 암호 시스템의 기호 (◆▲●■)와 1:1 매칭.

### 등장 흐름

| 회차 | 상태 |
|------|------|
| 1~2회차 | 없음 |
| 3회차 | 책상 모서리에 실루엣만. 탭 불가 |
| 5회차 | 윤곽 보임. 탭해도 반응 없음 |
| 7회차 | 완전히 나타남. 탭하면 다이얼 클로즈업. 기호들이 전부 ? 상태 |
| 코스메틱 전부 수집 후 | 올바른 기호로 맞추면 상자가 열림 |

### TRUE ENDING 해금 흐름

```
다이얼 상자 발견 (7회차) →
챕터 클리어 (일반 엔딩) →
코스메틱 팩 수집 (기호 조각 발견) →
4개 기호 완성 →
다이얼 상자 열림 →
창조자의 마지막 기억 공개 →
숨겨진 방 6 해금 →
TRUE ENDING
```

## 6. 게임 시작 직후 흐름

타이틀에서 카드 덱을 탭하면 카드를 집어드는 간단한 2D 애니메이션 → 화면 전환.

```
[배경: 어두운 서고 입구]

┌─────────────────────────────────┐
│ Elise                            │
│ "카드를 가져왔군요."              │
│ "여기선 카드로 길을 열어요.       │
│  이기면 앞으로 나아갈 수 있어요." │
└─────────────────────────────────┘
              ▼ 탭
         첫 번째 배틀
```

## 7. Path 구조

```
시작
 │
 ├─ ⚔  일반 배틀    → 카드 / 덱 강화
 ├─ 📦 기억 상자   → 힌트 조각 획득
 ├─ 🔥 캠프파이어  → 카드 강화 or 회복
 ├─ ⚔  중간 보스   → 큰 힌트 획득
 │
[방 자물쇠 화면] → 힌트로 자물쇠 해제
 │
 └─ 👁  수호자 배틀 → 다음 방으로
```

### 기억 상자 — 힌트 획득 방식

형식: 찢긴 일기 한 페이지, 흐릿한 사진, 녹음된 목소리(텍스트), 메모

직접적인 정답이 아닌 맥락 속에 녹아있는 힌트.

## 8. 카드 시스템

### 승패 조건 — 기억 역전 (Memory Inversion)

양쪽 모두 기억 조각 5개로 시작. 카드로 공격하면 수호자의 조각을 빼앗음. 수호자 조각 0 → 승리 / 플레이어 조각 0 → 패배.

### 자원 시스템

| 자원 | 역할 |
|------|------|
| 잉크 (Ink) | 카드를 내는 자원. 매 턴 3개 자동 충전. 턴 종료 시 소멸 |
| 망각 (Oblivion) | 강한 카드를 내기 위한 추가 자원. 덱에서 카드 1장을 영구 제거하면 잉크 +2 획득 |
| 기억 조각 (Fragment) | 순수 HP 역할. 잉크와 분리 |

### 망각 시스템

- 기본 잉크: 매 턴 3개
- 망각 1회: 덱에서 카드 1장 영구 제거 → 잉크 +2 (이번 턴만)
- 회귀 시: 망각된 카드 전부 복구
- 복구된 카드: "기억된 카드"로 강화 (비용 감소 or 공격력 증가)

### 슬롯

필드에 4개 슬롯. 카드를 슬롯에 내려놓는 방식.

### 흐릿한 기억 (Faded Memory)

Inscryption 다람쥐 카드에 해당하는 Recall만의 토큰.
- 잉크 0개, 공격 0, 방어 0
- 망각 재료로 사용 가능 (+2 잉크)
- "너무 오래되어 내용을 알 수 없는 기억."
- 스토리 연결: 나중에 이 카드들이 수호자들의 잃어버린 기억이었다는 암시

## 11. 말하는 카드 시스템

Inscryption의 Stoat처럼 플레이어에게 직접 말을 거는 카드. 회차마다 대사가 달라짐.

*주의: 구체적인 대사 내용은 카드 밸런스 테스트 및 게임 시스템 확정 이후 다듬을 것.*

### 길 잃은 기억 (Lost Memory)
- 시작 덱 포함. 방탈출 힌트 제공 역할.
- 회차가 쌓일수록 회귀를 인식하는 듯한 대사로 변화.

### 속삭임 (Whisper)
- 방 2 Path에서 획득.
- 수호자의 패턴이나 약점을 암시하는 역할.
- 우리 카드 시스템(잉크/망각/조각)에 맞는 언어로 다듬을 것.

### 오래된 메모 (Old Memo)
- 방 3+ 기억 상자에서 획득.
- 다이얼 상자와 코스메틱 오버레이 시스템으로 유도하는 역할.
- 매 사용마다 조금씩 다른 메시지.

### J의 카드 (J's Card)
- 방 2 기억 상자에서만 발견 가능.
- 내러티브 핵심 카드. J가 서고에 있음을 처음으로 확인.
- 회차마다 J의 기억이 점점 희미해지는 방향으로 대사 변화.

## 12. 수호자

### 관계도

```
창조자 (방 6)
    │
    └─ 노인 (방 4) — 창조자와 연결, 가장 오래됨

J ──┬─ Elise (방 1) — J의 선배, J가 앱 소개해줌
    ├─ Owen (방 2) — J와 같은 동아리
    └─ [방 5에서 직접 등장]

플레이어 ── Finn (방 3) — 플레이어의 친구, J 찾으러 먼저 들어옴
```

### Elise — 방 1 / 잉크 압박형
- **배경**: J의 선배. 취업 스트레스로 앱 시작. 잃어버린 것: 꿈.
- **스타일**: 매 턴 잉크를 최대로 써서 필드를 빠르게 채움. 약점: 조각을 빠르게 빼앗아 조기 종료.
- **회귀 대사**: 1회차 친절 → 3회차 "몇 번째죠?" → 6회차 "나처럼 되기 전에."

### Owen — 방 2 / 망각 축적형
- **배경**: J의 동아리 친구. 괴로운 기억을 지우려다 좋은 기억까지 잃음.
- **스타일**: 초반 약하다가 플레이어가 망각할수록 강해짐. 약점: 망각 없이 속전속결.
- **회귀 대사**: 1회차 침묵 → 3회차 "저도 그랬어요" → 6회차 "뭘 지우고 싶었는지도 모르겠어요."

### Finn — 방 3 / 에너지 트릭형
- **배경**: 플레이어의 친구. J를 찾으러 먼저 들어왔다가 갇힘.
- **스타일**: 잉크를 모았다가 버스트. 빈 턴을 노리면 유효. 약점: 모으는 동안의 빈틈.
- **회귀 대사**: 1회차 "누군가를 찾아서 온 거죠?" → 6회차 "...당신, 나 알아요?"

### 노인 — 방 4 / 모방형
- **배경**: 이름 없음. 창조자와 연결된 가장 오래된 수호자.
- **스타일**: 플레이어가 이번 런에서 쓴 카드를 그대로 모방. 약점: 예측 불가 혼합 전략.
- **회귀 대사**: 1회차 "그냥 돌아가요" → 7회차+ "서고가 당신을 눈치챘어요."

## 13. 친구 J의 등장 구조

| 방 | J의 흔적 | 플레이어의 감정 |
|----|---------|--------------|
| 방 1 | 기억 상자에서 J의 기억 파편. "여기서 나가야 한다"는 메모 | "J가 여기 있었던 건가?" |
| 방 2 | 케이스 안에 J의 물건. 입고일: 2주 전 | "J의 물건이 왜 전시되어 있지?" |
| 방 3 | J의 목소리 파편. "나 여기서 나갈 수 있을 것 같아—" 끊김 | "왜 못 나간 거지?" |
| 방 4 | 원본 기록에서 J 이름 + 상태: "수호자 전환 대기 중" | "내가 빨리 가야 해" |
| 방 5 | J가 직접 등장. 멍한 눈빛. 카드를 들고 있음 | 선택의 순간 |

### 방 5에서의 선택

```
J와 배틀      →  J의 남은 기억 가져옴  →  ARCHIVE ENDING
배틀 거부     →  J: "저 문으로 가면 안 돼"  →  FREEDOM ENDING
J 무시        →  LOOP ENDING
```

## 14. 방 구조

### 방 1 — 접수실
- 수호자: Elise / 자물쇠: 3개 / 분위기: 밝고 평범
- 문서 조각: 입고 기록부 / 드러나는 것: J의 이름 (날짜: 3주 전)

### 방 2 — 기억의 복도
- 수호자: Owen / 자물쇠: 4개 / 분위기: 박물관 같은 낯섦
- 문서 조각: 물건 태그 / 드러나는 것: J의 물건. 수호자의 이름표가 자기 케이스에

### 방 3 — 심문실
- 수호자: Finn / 자물쇠: 4개 / 분위기: 처음으로 불편해짐
- 문서 조각: 진술서 / 드러나는 것: 진술서에 플레이어 이름. 날짜는 어제

### 방 4 — 핵심 보관실
- 수호자: 노인 / 자물쇠: 5개 / 분위기: 무거워짐
- 문서 조각: 원본 기록 / 드러나는 것: 창조자도 갇혀있음. 7회차 이후 자물쇠 다시 잠힘

### 방 5 — 거울의 방 (분기점)
- 자물쇠 없음. 거울과 세 개의 문. J 등장.

### 방 6 — 창조자의 방 (TRUE ENDING 전용)
- 코스메틱 팩 전체 + 다이얼 상자 해제 시에만 진입 가능

## 15. 자물쇠 시스템

1. **교차 힌트**: A방 + B방 힌트 조합으로 C 자물쇠 해제 가능
2. **카드 자체가 힌트**: 카드의 이름, 능력치, 그림에 힌트 내포
3. **범위 추론**: 힌트 조합으로 범위를 좁힐 수 있게 설계
4. **수호자 실수**: 여러 번 틀리면 수호자가 배틀 중 힌트를 흘림

## 16. 엔딩 분기

| 엔딩 | 조건 | 내용 |
|------|------|------|
| **LOOP ENDING** | 힌트 부족한 채 끝까지 도달 | 탈출했다 생각했지만 다시 방 1 |
| **FREEDOM ENDING** | 모든 힌트 수집 + J와 배틀 거부 | 회귀 사이클 멈춤. 수호자들 해방 |
| **ARCHIVE ENDING** | J와 배틀해서 이김 | 플레이어가 새로운 수호자로 흡수 |
| **TRUE ENDING** | 코스메틱 전체 + 다이얼 상자 해제 | 창조자 해방. 회귀 사이클 완전히 끊음 |

캠퍼스. 낮. J가 웃으며:
> *"야, 요즘 캠퍼스에 이상한 앱 돌아다니더라. 너 봤어?"*

주인공만 기억하고 있음. 제목 화면으로 컷 — **RECALL**

## 18. 코스메틱 시스템

### 팩 구성

| 팩 | 가격 | 테마 | 다이얼 기호 |
|----|------|------|------------|
| **기본 팩** | 무료 | 낡은 양피지, 세피아톤 | ◇ (윤곽만) |
| **수호자의 기억** | ₩1,900 | 어둡고 붉은 필사본 | ◆ |
| **디지털 서고** | ₩1,900 | 글리치, 회로 패턴 | ▲ |
| **해방된 기억** | ₩1,900 | 따뜻하고 밝은 톤 | ● |
| **창조자의 흔적** | ₩2,900 | 시즌 한정 | ■ |

### 오버레이 암호 시스템

각 코스메틱 팩은 그 자체로 완성된 디자인. 올바른 순서로 겹쳤을 때만 아래 레이어의 기호가 비쳐 보임. 각 팩의 투명창은 디자인의 일부처럼 자연스럽게 위장.

올바른 순서로 모든 팩을 겹치면 4개 기호 (◆▲●■) 동시 가시화 → 다이얼 상자 코드 완성.

### 4단계 발견 레이어

1. **개인 발견**: 코스메틱 하나에 숨겨진 디테일. 눈 밝은 사람만 발견
2. **조합 발견**: 팩 2개 이상을 겹치면 뭔가 이어짐
3. **전체 완성**: 모든 팩을 겹치면 4개 기호 완성 → 다이얼 상자 코드
4. **게임 밖 연결**: 방 6 해금 후 ARG 요소 (미정)
